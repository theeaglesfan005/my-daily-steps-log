# My Daily Steps Tracker

A beautiful iOS app for tracking daily steps and fitness goals with Apple Health integration.

## Features

- View daily, weekly, monthly, and yearly step statistics
- Track workouts and activities
- Set custom step goals
- Beautiful visualizations and progress tracking
- Sync with Apple Health
- Override and adjust displayed step counts when needed
- Built-in tools to troubleshoot Health sync issues
- Manage workout display and resync from Health
- iCloud backup for your data
- Dark mode optimized
